# Webp-Converter
A testing application of the library "webp-converter", will convert all .webp files on the "arquivos" folder to .jpg

### Getting Started :computer:

```sh
git clone https://github.com/muNeves3/Webp-Converter
```

## Install :robot:

```sh
npm install
```

## Runing :rocket:

```sh
node index.js
```
